# Домашнее задание по темам "Инфраструктура", "Node.js", "Тесты"


## Установка
```
npm i
```

## Запуск сервера в Dev режиме
```
npm run server:dev
```
## Запуск сервера в Production режиме
```
npm run server:prod
```

### Dev-сборка(слежение за изменением проекта)
```
npm run watch
```

### Production-сборка
```
npm run build
```
